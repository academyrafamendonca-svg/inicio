# Academy Rafa Mendonça — Landing Page

Página de vendas estática (HTML + imagens). Pronta para GitHub Pages.

## Publicar no GitHub Pages

1. Cria um repositório novo (ex: `academy-rafa-mendonca`).
2. Faz upload de **todo o conteúdo desta pasta** (o `index.html` e a pasta `img/`).
3. No repositório: **Settings → Pages → Branch: `main` / `(root)` → Save**.
4. Ao fim de ~1 min a página fica em: `https://SEU-USER.github.io/academy-rafa-mendonca/`

## Domínio próprio (opcional)
Em **Settings → Pages → Custom domain**, coloca o domínio (ex: comprado na Amen.pt) e ativa **Enforce HTTPS**.

## Prévia de partilha (WhatsApp/Instagram)
